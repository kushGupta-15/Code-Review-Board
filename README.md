# Code Review Board

A collaborative code review platform where developers can share snippets, 
leave inline threaded comments, and see real-time presence of teammates.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Zustand, Prism.js
- **Backend:** Node.js, Express, Socket.io
- **Database:** PostgreSQL
- **Auth:** JWT + bcrypt

## Features
- Paste and share code snippets with syntax highlighting
- Inline threaded comments per line (GitHub-style)
- Real-time presence — see who's viewing a snippet live
- Workspace/team support
